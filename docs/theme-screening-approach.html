Grok API EquityQuery Theme Screening 新機能提案

テーマ別スクリーニング
× Grok駆動テーマ発見

「今どのセクターが熱い？」を Grok に聞いてから、そのテーマで割安株をスクリーニングする2段階パイプライン

スクリーニングを「条件入力」から「意図入力」へ。
ユーザーが「今熱いテーマで割安株を探して」と言うだけで、 Grok がトレンドテーマを検出し、EquityQuery がそのテーマ内で質の高い銘柄を絞り込む。 市場の空気を読んで投資候補を自動で届ける。

🔄
2段階パイプライン
Step 1
Grok でトレンド検出
X / Web を横断して今注目のセクター・テーマを自然言語で取得
Grok API
Step 2
テーマ名をマッピング
Grok の回答を themes.yaml のテーマキーに変換（ai / ev / defense など）
themes.yaml
Step 3
テーマ × プリセットでスクリーニング
テーマに対応する industries を EquityQuery に変換し、既存プリセット条件と AND 結合
EquityQuery
Step 4
テーマ解説つきで出力
「なぜ今このテーマか」の理由と銘柄スコアをセットで提示
Report
💬
Grok とのやり取りイメージ
▶ Claude → Grok に送るプロンプト
"What sectors or investment themes are trending in global markets right now?
Return as JSON array of theme names with brief reasons.
Focus on 3–5 themes that have clear momentum."
◀ Grok からの回答（例）
[
  { "theme": "AI/Semiconductor", "reason": "NVIDIA Blackwell需要爆発、AI CapEx $6500億超" },
  { "theme": "Defense", "reason": "欧州NATO予算拡大、地政学リスク継続" },
  { "theme": "Energy Transition", "reason": "再エネ投資再加速、IRA補助金執行継続" }
]
→ テーマキーに変換: ai defense renewable-energy

trending preset

新機能

情報源 X + Web（マクロトレンド）
検出軸 業界動向・マクロ・資金フロー
